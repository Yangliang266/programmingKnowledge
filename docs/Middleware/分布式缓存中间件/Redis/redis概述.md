## redis

### 是什么

#### 定义

> 远程字典服务，以kv的形式进行存储

#### 分类

> NoSQL

##### 特点

###### 表现形式上

> 非结构化数据，表和表之间不关联，扩展性强

###### 存储性能上

> 海量数据高并发读写，数据进行分布式存储

###### 表现形式

> 数据最终一致性
